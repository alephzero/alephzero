## Install Instructions

```sh
git clone https://github.com/alephzero/alephzero.git
cd alephzero
make install -j
```

## Ubuntu Dependencies

```sh
apt install g++ make
```

## Alpine Dependencies

```sh
apk add g++ linux-headers make
```
